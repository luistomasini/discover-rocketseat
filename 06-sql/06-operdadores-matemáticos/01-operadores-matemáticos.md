## + adição, - subtração, * multiplicação, / divisão, % módulo

```sql

SELECT * FROM aluno WHERE matricula = 1 + 1
SELECT * FROM aluno WHERE matricula = 1 - 1
SELECT * FROM aluno WHERE matricula = 1 * 1
SELECT * FROM aluno WHERE matricula = 1 / 1
SELECT * FROM aluno WHERE matricula = 4 % 3
-- resto 1

```
