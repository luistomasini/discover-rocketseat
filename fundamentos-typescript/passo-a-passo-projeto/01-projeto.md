# Projeto

Link do repositório:

<https://github.com/rocketseat-education/lista_presenca/tree/typescript>

Clonando o projeto:

`git clone https://github.com/rocketseat-education/lista_presenca.git`

Acessando diretório:

`cd lista_presenca`

Instalando as dependências:

`npm install`

Executando o projeto:

`npm run dev`
